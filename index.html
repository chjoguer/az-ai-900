🤖 Certificación AZ-AI-900

Azure AI Fundamentals

¿Qué es AZ-AI-900?

La certificación Microsoft Azure AI Fundamentals (AZ-AI-900) valida conocimientos fundamentales sobre conceptos de inteligencia artificial y servicios de Azure AI.

Áreas de Conocimiento

🧠 Conceptos de IA

Fundamentos de Machine Learning, Computer Vision, NLP y IA Conversacional

☁️ Azure AI Services

Azure Machine Learning, Cognitive Services y Azure Bot Service

🔍 Computer Vision

Análisis de imágenes, detección de objetos y reconocimiento facial

💬 Natural Language Processing

Análisis de texto, traducción y comprensión del lenguaje

🤝 IA Responsable

Principios éticos, privacidad y consideraciones de gobernanza

📊 Machine Learning

Tipos de ML, entrenamiento de modelos y evaluación

Recursos de Estudio

  • Azure Cognitive Services
  • Azure Machine Learning
